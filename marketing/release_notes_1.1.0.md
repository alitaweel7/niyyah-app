# Niyyah 1.1.0 — App Store submission copy

Submitted for review 2026-06-27 (build 1.1.0+8). Reuse / adapt for future updates.

## What's New in This Version

### English (U.S.)
```
• Niyyah is now fully available in Arabic, with complete right-to-left support — choose your language on first launch or in Settings.
• New Qibla compass: find the direction of the Kaaba from anywhere.
• Prayer tracker: log your five daily prayers right from the home screen.
• More accurate prayer times, tuned to your location and madhab.
• A more beautiful reading experience: the Bismillah now appears on its own line, and sequential reading continues exactly where you left off.
• Clearer progress: see your true journey through the Qur'an.
• Calmer notifications — fewer, more meaningful reminders each day, in your language.
• A refreshed, more peaceful design throughout, with a new dark mode.

Set your niyyah before you scroll.
```

## App Review note (paste into App Review Information → Notes)
```
Niyyah uses Apple's Screen Time / Family Controls APIs so a user can shield their own
chosen apps behind Qur'an reading, for personal self-discipline. The DeviceActivityMonitor
extension re-applies the user's own shields when a self-set unlock window ends. No
third-party or child monitoring is involved.
```
