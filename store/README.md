# store/ — App Store listing reference

`app_store_description.md` is the local reference copy of the App Store Connect listing
(description, keywords, category). It is **not** synced automatically — when you change the
listing in App Store Connect, update it here too, and vice versa.

Notes:
- The live listing is [Niyyah - Set Your Intention](https://apps.apple.com/app/id6761061944) (app ID `6761061944`).
- English only so far; an Arabic listing localization is on the roadmap (the app itself is fully bilingual).
- The "FREE FEATURES / gate up to 3 apps" framing in the original copy was never enforced in the
  app — today everything is free and unlimited. Revisit this file when Niyyah Plus ships.
- Release-specific copy (What's New, review notes) lives in `marketing/release_notes_<version>.md`.
